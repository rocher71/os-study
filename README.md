## 스터디 방식

매주 혼공컴운 책을 공부하거나, 인프런/유튜브의 공식 강의를 듣고 서로 공부한 내용을 발표하고 퀴즈를 푼다.

+ 강의 듣고 깃허브에 정리한다
+ 정리한 내용을 발표(라 쓰고 대화라 읽는다)한다
+ 강의를 들으면서 어려웠던 부분을 이야기한다
+ 핵심 개념들을 기반으로 퀴즈를 내고 맟혀본다
